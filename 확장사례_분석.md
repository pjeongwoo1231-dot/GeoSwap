# Geo-Swap 확장 사례 분석 — "카자흐 프로토타입 → 전 세계 플랫폼"

> 기획서 [발전가능성/확장성] 섹션 및 2차 발표용 재료.
> 핵심 메시지: **Geo-Swap은 카자흐스탄 단일 사례가 아니라, "강대국이 분할 점유한 분쟁·내륙 산지" 전반에 적용되는 플랫폼이다.**

---

## 0. 모델이 적용되는 산지의 4대 조건

카자흐스탄 모델의 본질을 일반화하면, 다음 4조건 중 다수를 충족하는 산지면 동일 구조가 성립한다.

1. **운송 취약** — 내륙국(파이프라인 인질) 또는 분쟁 해협/항로 통과 필요
2. **강대국 분할 점유** — 미·러·중 등이 유전 지분/운영권을 나눠 가짐
3. **메이저(특히 미국)가 실질 운영** — 신용도 높은 운영주체 → RWA/스왑 상대방 리스크 해소
4. **품질-지정학 괴리** — 원유 품질은 좋은데 지정학 때문에 저평가(할인) 거래

→ Geo-Swap은 이 산지의 **'권리(RWA)'를 확보하되 물리적 인도는 안전 산지와 스왑**하여, 운송 리스크를 금융으로 치환하고 할인(Basis)을 차익화한다.

## 1. 확장 대상국 (적합도 순)

### ⭐⭐⭐⭐⭐ 이라크 (남부 유전) — "미·러·중 3분할의 교과서"
- **강대국 분할:** ExxonMobil(미, West Qurna-1) · **Lukoil(러, West Qurna-2)** · **CNPC/PetroChina(중, Rumaila·Halfaya)** · BP — 미·러·중이 **각각 대형 유전을 실제 운영**. 카자흐의 "동일 비율 분할" 서사가 가장 문자 그대로 성립.
- **지정학:** 호르무즈 해협 의존 + 내부 정정 불안 + 이란 영향권.
- **품질:** Basra 중질·고황(중동형) → 국내 정유설비 친화.
- **우리 모델 적용:** 이라크 RWA 확보 → 호르무즈 우회 불가 시 미·UAE 등과 스왑.
- **데이터 보유:** ✅ K-SURE 등급 7(최고위험)·GPR(MiddleEast)·수입량·API27.6/황3.08 전부 보유 → **대시보드에서 즉시 시연 가능.**

### ⭐⭐⭐⭐⭐ 가이아나 — "가장 핫한 신흥 산지"
- **강대국 분할:** ExxonMobil(미, 45%) · Hess · **CNOOC(중국, 25%)** 공동 운영(Stabroek 광구).
- **지정학:** **베네수엘라와 에세키보(Essequibo) 영토분쟁** — 무력충돌 위험.
- **품질:** Liza 초경질·저황(고품질) → 카자흐처럼 "고품질인데 지정학 할인" 구조.
- **우리 모델 적용:** 가이아나 권리 확보 → 분쟁 격화 시 미걸프·브라질 원유와 스왑.
- **데이터:** GPR(Americas)·품질 추정 가능. (수입 실적은 적으나 "미래 확장처"로 서술)

### ⭐⭐⭐⭐ 투르크메니스탄 — "카자흐의 쌍둥이(내륙국)"
- **강대국:** 가스 매장 세계 4위, 수출의 대부분이 **중국 파이프라인(CNPC)에 인질**.
- **지정학:** 완전 내륙국 + 카스피해 영유권 + 단일 고객(중국) 의존.
- **우리 모델 적용:** 가스 버전 Geo-Swap(LNG 스왑)으로 확장 — 한국가스공사 데이터와 직결.

### ⭐⭐⭐⭐ 앙골라 — "중국 자금 + 미국 메이저"
- **강대국:** **중국의 자원담보 차관**으로 깊이 연결 + ExxonMobil·Chevron·TotalEnergies 운영.
- **지정학:** 정치 리스크 + 장거리 해상 운송.
- **데이터:** ✅ K-SURE 6·GPR(Africa)·수입량·품질 보유.

### ⭐⭐⭐ 베네수엘라 — "제재 자산의 금융화"
- **강대국:** Chevron(미)이 미 정부 라이선스로 통제, 제재 대상.
- **핵심 논리:** 제재국과 직접 거래 ❌ → **미국 메이저의 합법 자산만 RWA 편입**(컴플라이언스 방패).
- **데이터:** ✅ K-SURE 7·GPR(Venezuela)·수입량·API13.9/황3.6(초중질) 보유.

### ⭐⭐⭐ 아제르바이잔 — "러시아 우회 설계의 원형"
- **강대국:** BP 주도(ACG 유전), **BTC 파이프라인이 러시아·이란 우회**(터키 경유) — 지정학 우회의 실제 사례.

## 2. 전략적 함의 (기획서 문구용)

> "본 플랫폼은 카자흐스탄을 **프로토타입**으로 설계되었으나, 그 본질은 **'강대국이 분할 점유한 운송 취약 산지'**라는 구조적 패턴에 있다. 이라크(미·러·중 3분할), 가이아나(미·중 공동, 영토분쟁), 투르크메니스탄(내륙·중국 인질) 등 **전 세계 6개국 이상**에 동일 모델이 즉시 적용 가능하며, 이는 단일 아이디어가 아닌 **확장형 에너지 금융 플랫폼**임을 의미한다."

## 3. "말이 아니라 작동"의 증거 (차별화 포인트)
- 위 국가 중 **이라크·앙골라·베네수엘라**는 이미 본 플랫폼 데이터(K-SURE 등급·AI-GPR 지정학지수·수입량·원유품질)에 포함 → **대시보드 석유 환율 계산기에서 '위험산지'로 선택 시 즉시 스왑비율 산출.**
- 즉 "확장 가능성"이 **슬라이드 문구가 아니라 라이브 데모로 증명**됨. (심사 2차 발표 시 직접 시연)

## 4. (옵션) 대시보드 확장 위젯 아이디어
"확장 대상국 비교" 패널: 이라크·앙골라·베네수엘라·카자흐의 [K-SURE 등급 / GPR / 품질 / 현재 스왑비율]을 한 표로 → 플랫폼의 다국가 적용성을 한눈에.
