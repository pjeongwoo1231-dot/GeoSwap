## 수정 2. 석유 환율 계산기에 "국가별 지정학 할인" 반영 (핵심)

**문제:** 현재 "카자흐스탄(Brent) 1배럴 = 사우디(Dubai) 0.9986배럴" → 거의 1:1이라 의미 없음.
**원인:** 국가 원유에 벤치마크 가격을 **그대로** 써서, 카자흐 같은 고위험 원유의 **실제 거래 할인(Basis)**이 반영 안 됨.

### 2-1. `engine.py` 상단에 할인율 테이블 추가
```python
# 벤치마크 대비 할인율(양수 = 그만큼 싸게 거래됨).
# 실제 원유 Basis(CPC Blend, Urals, Merey 등)의 통상 수준을 반영한 '조정 가능한 가정값'.
# 추후 한국무역보험공사(K-SURE) 국가위험도 / Argus·Platts 실측 Basis로 대체 예정. # TODO(v2)
GEO_DISCOUNT = {
    # 내륙국·제재·분쟁 인접 → 큰 할인
    "카자흐스탄": 0.07,   # CPC Blend, Brent 대비 통상 5~8% 할인 (러 영토 파이프라인 리스크)
    "러시아":     0.25,   # Urals/ESPO, 제재 후 대폭 할인
    "베네수엘라": 0.25,   # Merey 중질유 + 제재
    # 서아프리카 등 약한 할인
    "나이지리아": 0.02,
    "앙골라":     0.02,
    "콩고":       0.03,
    "가봉":       0.02,
    # 중동(사우디/UAE/쿠웨이트/이라크/카타르/오만), 미국, 캐나다 등 = 벤치마크 par → 0
}
# 없는 국가는 0.0
```

### 2-2. 유효 가격 함수 (정석 산식)
```python
def effective_price(country, benchmark_price):
    """국가 원유의 유효가격 = 벤치마크가격 × (1 − 지정학할인)"""
    d = GEO_DISCOUNT.get(country, 0.0)
    return benchmark_price * (1 - d)

def swap_rate(country_a, bench_price_a, country_b, bench_price_b):
    """A 1배럴 = B 몇 배럴 (지정학 할인 반영)"""
    return effective_price(country_a, bench_price_a) / effective_price(country_b, bench_price_b)
```
- 즉 `SwapRate(A→B) = [P_benchA × (1−d_A)] / [P_benchB × (1−d_B)]`.
- 기존 벤치마크→벤치마크(국가 미선택) 모드가 있으면 그쪽은 d=0으로 그대로.

### 2-3. 계산기 탭(탭4) UI 수정
1. 결과 숫자 아래에 **계산 근거 한 줄** 표시(투명성 = 신뢰도):
   ```
   Brent $72.9 × (1 − 7% 지정학할인)  ÷  Dubai $73.0 × (1 − 0%)  =  0.93
   ```
   → st.caption 등으로.
2. **"지정학 할인율 직접 조정" 슬라이더** 추가:
   - 유종 A의 할인율을 `GEO_DISCOUNT` 기본값으로 초기화한 `st.slider(0.0, 0.30, 기본값, 0.01)`.
   - 슬라이더 움직이면 스왑비율·시계열이 **즉시 반영**되게.
   - 이게 v2(지정학 가중치) 슬롯이 살아있음을 보여주는 인터랙티브 데모.
3. 할인 반영 후 기대 결과(검증용): 카자흐(Brent,−7%) → 사우디(Dubai,0%) ≈ **0.93** 근처가 나와야 정상. (1.0 그대로면 반영 안 된 것.)
