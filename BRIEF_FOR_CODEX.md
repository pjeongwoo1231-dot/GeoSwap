> 너(코딩 에이전트)는 이 프로젝트의 **데이터 구조·산식·화면 구성을 이미 확정된 사양**으로 받아서, Streamlit 웹 대시보드를 구현/배포하는 역할이다. 데이터 구조를 임의로 추측하지 말고 아래 명세를 그대로 따를 것.

## 0. 프로젝트 한 줄 정의

**위험한 산지의 원유를 직접 실어오지 않고, 그 '권리'를 안전한 산지 원유와 교환(스왑)한다. 교환 비율 = AI가 산출하는 '석유 환율(Petroleum Swap Rate)'.**
이 대시보드는 **산업부 공공데이터(한국석유공사 등)** 로 그 석유 환율과 한국의 원유 수입 구조/지정학 리스크를 시각화하는 **작동하는 웹 서비스**다.

- 제출처: **제14회 산업통상부 공공데이터 활용 아이디어 공모전 — 제품 및 서비스 부문**
- 제품·서비스 부문은 **"심사위원이 접속 가능한 서비스 링크" 제출이 필수**다. → 최종 목표는 **Streamlit Community Cloud 배포 링크**.

## 2. 데이터 파일 (이미 `data/`에 있음) — ⚠️ 파싱 주의점 필독

세 파일 모두 **UTF-8**, 단위 주의. 헤더가 한국 통계포털 특유로 지저분하니 아래대로 정제할 것.

### 2-1. `data/국가별_원유수입.csv`  (단위: **천 배럴**, **연도별** 2020~2024)
- 헤더: `"15318AA6 국가별(1)", 15318AA6 국가별(2), Y2020 2020, Y2021 2021, Y2022 2022, Y2023 2023, Y2024 2024`
- 구조: **계층형**.
  - col0 = 대륙 (예: `"15318AA6AA 아시아"` → 코드 + 공백 + 이름)
  - col1 = 국가 (예: `"15318AA6AAAB 필리핀"`) **또는** `"소계"`(대륙 소계) / 최상단은 col0=`"15318AA600 합계"`,col1=`"소계"`(전체합계)
  - 연도 컬럼명 `Y2020 2020` 등 → 연도 `2020`만 추출.
- **정제 규칙:**
  - 국가 행만 남기기: `col1 != "소계"` 인 행. (소계/합계 행은 검증용으로만 분리 보관)
  - `국가명 = col1.split(" ", 1)[1]`  (멕시코는 코드 끝에 숫자 있어도 첫 공백 기준 split이면 OK)
  - `대륙명 = col0.split(" ", 1)[1]`
  - 빈 셀 = 그 해 수입 없음 → `0` (또는 NaN 후 0 fill)
  - 값은 정수(천 배럴).
- **결과 형태(권장 tidy):** `연도 | 대륙 | 국가 | 물량_천배럴`
- ⭐ **데이터 속 핵심 스토리(반드시 시각화로 부각):**
  - **러시아**: 2020=46,929 → 2021=53,748 → 2022=20,983 → **2023·2024=0** (전쟁·제재로 수입 중단)
  - **카자흐스탄**: 2022=47,599 → 2024=11,529 (급감 — 내륙국 파이프라인 리스크)
  - **미국**: 104,409 → 168,411 (급증 — 수입 다변화)

### 2-2. `data/유질별_원유수입.csv`  (단위: **천 배럴**)
- 헤더: `"유질별(1)", 2024.01, 2024.02, ... 2024.12, 2020, 2021, 2022, 2023`
- 행: `합계`, `경질유`, `중(中)질유`, `중(重)질유`
- 유질 매핑: **경질유=light**, **중(中)질유=medium**, **중(重)질유=heavy**
- **정제 규칙:**
  - 연도별 합계: 2020~2023은 해당 컬럼 직접 사용.
  - **2024년 합계 = 2024.01 ~ 2024.12 열 합산.**
  - `합계` 행은 검증용.
- **결과 형태:** `연도 | 유질(light/medium/heavy) | 물량_천배럴`  + (옵션) 2024 월별 시계열.

### 2-3. `data/국제유가.csv`  (단위: **$/배럴**, **월별** 2020.01~2024.12)
- 헤더: `월, 유종, Dubai, Brent, WTI, Oman`
- `월` 열: `"20년 01월"` 처럼 **연도 prefix가 1월 행에만** 붙고 이후엔 `"02월"`만 나옴. `유종` 열은 비어있음(무시).
- **정제 규칙:**
  - 연도 **forward-fill**: `"20년"`→2020, `"21년"`→2021 … (2자리 연도 → 2000+).
  - `"01월"`→month=1.
  - 4개 유종 가격은 float.
- **결과 형태:** `연월(YYYY-MM) | Dubai | Brent | WTI | Oman` (long으로 풀어도 됨)
- 연도별 평균이 필요하면 `groupby(연도).mean()`.

> ⚠️ 단위 불일치 주의: 물량은 **천 배럴**, 가격은 **$/배럴**. 금액 환산 시 ×1000 필요.

## 3. 석유 환율(Petroleum Swap Rate) 산식 — 핵심 로직 (`engine.py`)

### 3-1. 기본 정의 (v1 — 지금 구현)
두 유종 A, B 사이의 스왑 비율:

```
SwapRate(A→B, t) = P_A(t) / P_B(t)
```
- 의미: **"A 1배럴 = B 몇 배럴"** (가치 기준).
- 예: WTI=$74.82, Dubai=$72.87 → WTI→Dubai = 74.82/72.87 = **1.027** (WTI 1배럴 = Dubai 1.027배럴).
- 4개 벤치마크(Dubai/Brent/WTI/Oman) 전체 쌍에 대해 **월별/연도별 스왑비율 행렬**을 만들 것.

### 3-2. 국가→벤치마크 매핑 (국가별 원유에 가격을 입히기 위해)
국가별 데이터엔 가격이 없으므로, 각 국가 원유를 벤치마크로 프록시:
- 미국·캐나다·멕시코·브라질·에콰도르·콜롬비아 → **WTI**
- 중동(사우디·UAE·쿠웨이트·이라크·카타르·오만·중립지대) → **Dubai/Oman**
- 유럽·아프리카·러시아·카자흐스탄 → **Brent** (CPC 블렌드 등 Brent 연동, 할인 적용)
- (이 매핑 테이블은 `engine.py` 상단 dict로 두고 쉽게 수정 가능하게.)

### 3-3. 확장 산식 (v2 — 슬롯만 미리 뚫어둘 것, 기본값은 중립)
```
AdjustedSwapRate(A→B) = (P_A / P_B)
                        × (1 + geoDiscount_A − geoDiscount_B)   # 지정학 가중치
                        × gradeAdj                              # 유질 보정
                        × freightFactor                         # 운임 절감
```
- `geoDiscount`: 국가별 지정학 할인율. **출처 예정 = 한국무역보험공사(K-SURE) 국가위험도/국가신용등급.** 지금은 dict 기본값 0.0, **TODO 주석**으로 자리만.
- `freightFactor`: 항로 거리 기반. 지금은 주요 항로 거리 몇 개 하드코딩(예: 카자흐(흑해)→한국 vs 중동→한국). TODO 주석.
- `gradeAdj`: 유질별(경질/중질) 차이. 유질별 데이터 + 벤치마크 스프레드로 산출. v1.5.
- **중요: v1에서는 이 조정 항들이 1.0/0.0이라 기본 스왑비율과 동일하게 나와야 한다. 슬롯만 만들어 둘 것.**

## 4. 대시보드 화면 구성 (app.py — 탭 5개)

### 탭 1. 📊 원유 수입 구조 (국가별 데이터)
- 연도 슬라이더(2020~2024).
- 대륙별/국가별 수입 비중 (treemap 또는 bar).
- **수입 집중도 HHI 지수** 연도별 추이 (= Σ(국가 점유율)², 높을수록 편중). 라인차트.
- **중동 의존도(%)** 연도별 추이.
- ⭐ **고위험국(러시아+카자흐스탄) 노출 비중** 연도별 추이 → 급감 강조(주석/annotation).

### 탭 2. 🛢️ 유질 구성 (유질별 데이터)
- 경질/중질/중질 비중 연도별 추이 (stacked area).
- 2024 월별 추이(옵션).

### 탭 3. 💵 국제유가 & 스프레드 (유가 데이터)
- Dubai/Brent/WTI/Oman 월별 가격 라인차트.
- 유종 간 스프레드: **Brent−WTI**, **Brent−Dubai** 추이.

### 탭 4. ⭐ 석유 환율 계산기 (핵심 기능)
- 드롭다운 2개: 유종 A(보유/위험 산지) ↔ 유종 B(인도/안전 산지).
- 선택 시: **스왑비율 시계열**(월별) + **현재 비율 큰 숫자**로 표시. (예: "카자흐(Brent) 1배럴 = Dubai 1.04배럴").
- 슬라이더(옵션): 지정학 가중치/운임 조정 → AdjustedSwapRate 즉시 반영(슬롯 동작 데모).
- "이 스왑으로 절감되는 운송거리/탄소" 표시 칸 (v2 채움).

### 탭 5. 🌍 지정학 리스크 (v2 슬롯)
- "한국무역보험공사 국가위험도 데이터 연동 예정" placeholder + 현재는 수입량 변동성 기반 임시 리스크 지표라도 표시.

> 상단에 항상 **데이터 출처 푸터** 표시: `데이터: 산업통상부(한국석유공사) 국가별·유질별 원유수입 [KOSIS, tblId TX_31801_A008] / 국제유가 [페트로넷]`
